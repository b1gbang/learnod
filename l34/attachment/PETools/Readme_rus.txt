1. Описание программы
-----------------------------------
Полнофункциональная   утилита   для   работы   с   PE/PE+(64bit)  файлами.
Включающая в себя:  редактор PE файлов,  Task  Viewer,  оптимизатор  Win32
PE файлов, детектор компилятора/упаковщика и многое другое.

Основные возможности программы:
+ Task Viewer
  - Дампы процессов
    - Dump Full
    - Dump Partial
    - Dump Region
  - Возможность делать дампы .NET CLR процессов
  - Автоматическое снятие защиты "Anti Dump Protection"
  - Изменение приоритета процесса
  - Завершение работы процесса
  - Загрузка процесса в PE Editor и PE Sniffer
  - Generic OEP Finder
  
+ PE Sniffer
  - Определение типа компилятора/упаковщика
  - Возможность обновления базы сигнатур
  - Возможность сканирования директорий
  
+ PE Rebuilder
  - Оптимизация PE файла
  - Изменение базового адреса PE файла
  
+ PE Editor
  - Редактирование DOS заголовка
  - Поддержка нового формата PE+(64bit)
  - Корректирование CRC
  - Просмотр и редактирование таблиц импорта/экспорта

ВНИМАНИЕ: Дизайн очень похож на LordPE (от Yoda) - это сделано  специально,
чтобы люди, привыкшие к LordPE смогли легко перейти на нашу программу, ведь
наша намного лучше по функциональности!

3. Настройки
-----------------------------------
[ General ]
 - Always on top                     - Поверх всех окон
 - Restore last directory on startup - Сохранение и восстановление последней директории
 - Restore last main window position - Сохранение и восстановление размеров окна
 - Automatically check for updates   - Автоматическая проверка обновлений (каждый день)

 Drag'n Dropped files are... - При Drag'n Drop, файл будет загружен в,...
   - ...for the PE Editor      редактор PE файлов
   - ...for the PE Rebuilder   PE Rebuilder
   - ...for the PE Sniffer     PE Sniffer

[ PE Rebuilder ]
 - Register Shell Extension                - В контекстное меню системы, будет добавлен
                                             пункт - Rebuild PE
 - Dump fix            - Корректирование PE файла, после дампа из памяти
                         (необходимо для работоспособности файла после дампа из памяти)
 - Wipe Relocation     - Удаление релоков
   - ... in Dll        - удалять релоки из Dll
 - Validate PE         - Корректирование заголовков, для работоспособности
                         программы под Win2k
 - Rebuild PE          - Оптимизация файла
   - ... save overlay  - оверлей будет сохранён
                         (необходимо при оптимизации инсталляторов и SFX архивов)
   - File Alignment    - установить File Alignment (по умолчанию 0x200)

 - Bind Imports        - объединение импортов
 - Change ImageBase to - Изменение базового адреса файла (только при наличии релоков)

[ Task Viewer ]
 - Full Dump: paste header from disk       - Заменять PE заголовок копией с диска
 - Full Dump: paste import table from disk - Заменять таблицу импорта копией с диска
 - Full Dump: fix header                   - Корректирование PE файла
 - Full Dump: Force RAW mode               - Принудительная оптимизация дампа процесса
                                             (Используйте тогда, когда PE Tools падает)
 - Full Dump: rebuild image                - Оптимизация файла
 - Agreement required for terminations     - Подтверждение завершения работы процесса
 - Delete temp files                       - Удаление временного файла, после загрузки
                                             дампа в PE Editor
 - Display SoftICE status                  - Отображение статуса SoftICE

[ PE Editor ]
 - Register Shell Extension                - В контекстное меню системы, будет добавлен
                                             пункт - Load into PE Editor.
 - Create backup copy (BAK-file)           - Создание резервной копии файла
 - Read Only                               - Открывать все файлы - только для чтения
 - Section Table: autofix SizeOfImage      - Автоматическое корректирование SizeOfImage
 - Optional Header: autofix Checksum       - Автоматическое корректирование Checksum
 - Optional Header: autofix SizeOfHeaders  - Автоматическое корректирование SizeOfHeaders

[ PE Sniffer ]
 - Register Shell Extension                - В контекстное меню системы, будет добавлен
                                             пункт - Load into PE Sniffer

[ Break & Enter ]
 - Register Shell Extension                - В контекстное меню системы, будет добавлен
                                             пункт - Break & Enter

4. Использование командной строки
-----------------------------------
PETools.exe:

/PEEDITOR pefile.exe
Загрузка файла в PE Editor.
Будет отображено ТОЛЬКО окно редактора!

/PESNIFFER pefile.exe
Загрузка файла в PE Sniffer.

/REBUILDPE pefile.exe
Загрузка файла в Rebuild PE.

/BREAKENTER pefile.exe
Загрузка файла в "Break & Enter".
